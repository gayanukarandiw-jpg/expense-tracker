# 🏦 Expense Tracker

A modern, user-friendly desktop application to track your daily expenses with a beautiful dark-themed interface.

Built with **CustomTkinter** and **SQLite**.

---

## ✨ Features

- **Add Expenses** with title, amount, and category
- **Smart Auto-Suggestion** for categories based on description
- **View All Records** in a clean tabular format
- **Total Expense** summary
- **Category-wise Summary**
- **Visual Bar Chart** (using Matplotlib)
- **Delete Expenses**
- **Persistent Storage** using SQLite database
- Dark modern UI with smooth experience

---

## 🛠️ Technologies Used

- **Python**
- **CustomTkinter** (Modern Tkinter)
- **SQLite3** (Database)
- **Pandas** (Data analysis)
- **Matplotlib** (Visualization)
